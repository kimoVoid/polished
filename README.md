# Polished for b1.7.3
The everything QOL mod for Ornithe gen2.
<br>I gotta add more info to this lmao this is a continuation of [Beta QOL](https://github.com/kimoVoid/BetaQOL) which is also a general QOL mod for beta.

## Notable features
- Fixed the sounds the game downloads using BetaCraft proxy.
- Implemented Inventory Tweaks with a few changes to make it closer to vanilla.
- Modern debug screen.
- Front-facing perspective in F5.
- Multiplayer list with pinging.
- Zoom with animation and scrolling settings.
- Reduced about 300ms of ping on servers + TCPNoDelay enabled.
- Increased the auto-save timer to remove lag spikes.
- Infinite chat history.
- Option to allow click-mining on servers.
- Player list (tab) for servers running the mod.

This list will be updated properly, for now you can just check the mixins and figure it out.

## Dependencies
- [Silk Config](https://github.com/kimoVoid/silk-config) for in-game settings.
- [Ornithe Standard Libraries](https://modrinth.com/mod/osl) 0.18.0 or newer.

## Usage
1. Grab the Ornithe gen2 instance: https://ornithemc.net/ornithe-installer-rs/?prism&minecraft-version=b1.7.3&gen=2
2. Import the instance on Prism.
3. Get the latest artifact from here: https://nightly.link/kimoVoid/polished/workflows/build/main/Artifacts.zip
4. Inside the artifacts you will find a jar file, simply place it in your mods folder.

## Mod recommendations
- [Reframed](https://github.com/kimoVoid/reframed) for modern LWJGL 3 and overall better experience.
